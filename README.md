# 🤖 AI Araçları Paneli

Yapay zeka araçlarını (ChatGPT, Claude, Midjourney vb.) tek ekranda
listeleyen basit bir web paneli. Yazılım eğitimi 1. hafta projesi.

## Nasıl açılır?

Özel bir kurulum gerekmez:

1. `index.html` dosyasına çift tıkla (tarayıcıda açılır).
2. Sayfada "Deneme Kartı" görünüyorsa her şey çalışıyordur.
3. JavaScript testini görmek için `F12` > **Console** sekmesine bak.

## Dosyalar

| Dosya | Görevi |
|-------|--------|
| `index.html` | Sayfanın iskeleti (yapı) |
| `style.css` | Görünüm (renk, düzen, kartlar) |
| `script.js` | Davranış (arama, filtre — ileride) |
| `README.md` | Bu tanıtım dosyası |
